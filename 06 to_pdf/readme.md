# to_pdf

PDFファイルを作成するマクロです。  
エクセルの標準機能を使っています。  

inフォルダにPDFにしたいファイルを入れて、マクロを実行すると作成されます。

macro_name       | work                             | remarks
-----------------|----------------------------------|-------------------------
main_to_pdf      | シートごとにPDFにします          | 表示されているシートが対象です。
main_to_pdf_book | ブック全体をひとつのPDFにします  |
