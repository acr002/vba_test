# SPSSシンタックス作成

エクセル関数を使用し、SPSSシンタックスの元になるテキストデータを作成します。

## flow
SPSSに読み込ませるデータとシンタックスを作成していきます。
シンタックスの問Noとデータのヘッダが同じになるようにします。(並び・文言)
シンタックスの行数と、データの列数は同一になります。

### syntax
シンタックス作成用フォーマット.xlsxをしようします。
SAとRAはそれぞれ１設問につき、1行作成します。
SAはカテゴリを右に連ねて記述してください。
RAにカテゴリは不要です。
MAはカテゴリ分のレコードが必要です。該当するカテゴリを右にひとつだけ記述します。

1. 入力フォーマットに情報を入力 (入力サンプルシートを参考)
2. 出力シートの黄色の着色３列をコピー
3. テキストエディタに貼付
4. 固定になっているヘッダの３列分を削除
5. タブ(\t)を改行(\n)に置き換え
6. 複数の改行(\n\+)を一つの改行(\n)に置き換え
7. 余分な改行(\n)を削除

### data
csvのローデータを作成します。
ヘッダは1行にして、2行目からはデータになるようにしてください。

### SPSSに読み込ませる
SPSSを起動し、「ファイル - 開く - データ」からcsvのローデータを読み込ませます。
数値データが表に追加されます。
「ファイル - 開く - シンタックス」から作成したシンタックスを読み込ませます。
ウィザードが開くので、「実行 - 全て」を実行します。
シンタックスが反映されて、数値が文言に変わります。
savで保存して完了です。
